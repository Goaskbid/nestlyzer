# GitHub Pages deployment

1. Extract this ZIP.
2. Open the extracted folder.
3. Upload every file and folder inside it to the GitHub repository root.
4. Confirm `index.html` is about 18 MB and contains full HTML.
5. Confirm `CNAME` contains only `nestlyzer.com`.
6. In GitHub Settings, Pages, publish from `main` and `/root`.
7. Set the custom domain to `nestlyzer.com`.
8. Keep Namecheap DNS pointed at GitHub Pages.
