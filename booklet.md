# Methodology

Nestlyzer compares municipalities using visible profile weights, map scope, sub-rankings, public data and confidence. Switzerland currently has the deepest official-data module. European countries use the same municipality-level machine, with country-depth status shown in the app.

Confidence filters weak-data places. It never boosts scores.

Photos, coats of arms, flags, contacts, advertising, public text, image captions and cross-border context never affect scores.
